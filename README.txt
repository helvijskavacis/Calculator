# Python Command-Line Calculator

This is a simple, interactive command-line calculator written in Python. 
It supports basic arithmetic operations, exponentiation, and square roots. 
The calculator also keeps a history of your calculations in a local text file.

---

## Features

- Addition  
- Subtraction  
- Multiplication  
- Division (with zero-division handling)  
- Exponentiation  
- Square root (with negative number handling)  
- Saves all calculations to a file (`Calculations.txt`)  
- Option to view calculation history at the end of each session

---

### Prerequisites

- Python 3.x

### Running the Program

In your terminal or command prompt, run:

```bash
python 2nd_calc.py
